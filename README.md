# GigFinder - Local Event Discovery

A simple, static website to discover local gigs and events, similar to Hopmon.

## Features
- 🎵 Browse upcoming events
- 🔍 Search by event name, venue, or city
- 🏷️ Filter by genre and date
- 📱 Fully responsive design
- ⚡ Fast, no backend required

## How to Run Locally
1. Clone this repo
2. Open `index.html` in your browser
   - Or use a local server: `npx serve` or `python3 -m http.server`

## Deploy to GitHub Pages
1. Push code to GitHub
2. Go to **Settings > Pages**
3. Select branch `main` and folder `/ (root)`
4. Click **Save**

## Add Your Own Events
Edit `events.json` with your event data. Use real image URLs and ticket links.
