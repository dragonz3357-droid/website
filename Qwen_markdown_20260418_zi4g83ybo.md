# 🐾 Hopmon Game

A lightweight, mobile-friendly browser hopping game built with vanilla HTML5 Canvas + JavaScript.

## Features
- 🎮 Tap/Click or press Space to hop
- 📈 Procedurally generated platforms
- 🪙 Collectible orbs for bonus points
- 📱 Responsive & works on all devices
- ⚡ Zero dependencies, instant load

## How to Play
1. Open `index.html` in any modern browser
2. Tap/Click or press `Space` to hop
3. Avoid falling off platforms & collect orbs!

## Deploy to GitHub Pages (Free)
1. Push this repo to GitHub
2. Go to **Settings > Pages**
3. Select branch `main` and folder `/ (root)`
4. Click **Save** → your game will be live at `https://yourusername.github.io/hopmon-game/`

## Customization
- Change colors in `game.js` (`player.color`, `platforms` colors)
- Adjust difficulty by modifying `speed` and `spawnPlatform()` logic
- Replace canvas drawings with sprites by loading images in `game.js`
